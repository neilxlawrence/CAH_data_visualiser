The web application can be accessed via the shiny servers at the following URL: https://endocrinology.shinyapps.io/cah_data_visualiser/ 

This application is in prototype form, and should not be used with real patient data. 

Instructions to use the app are found embedded within the user interface. 

practice_upload_data.csv can be used to load data into the application. This is mock data, and does not contain any patient identifiable material.
